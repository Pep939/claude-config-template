# Coordination roles

Drop your own role `.md` files here. Each file defines one role on the `/coord`
multi-session bus: its job, hard rules, and how it coordinates with other roles.

See the included roles (`coder.md`, `reviewer.md`, `tester.md`, etc.) for the
format ‚Äî a `# Role: Name` heading, then `## Your job`, `## Hard rules`, and
`## Coordinate via` sections. These generic roles are ready to use as-is.
